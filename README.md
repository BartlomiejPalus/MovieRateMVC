# MovieRateMVC
MovieRateMVC is a web application built with ASP.NET Core MVC for managing movies. It allows users to add, edit, delete, browse, and rate movies. Users can filter movies by various criteria, and ratings are used to calculate average scores.

## Features
- Add, edit, and delete movies
- Browse movies with pagination
- Filter movies by criteria (e.g., title, director, genre)
- Rate movies and calculate average ratings
- User registration and login
- Data validation in forms

## Technologies
- C# / .NET 8
- ASP.NET Core MVC
- ASP.NET Identity
- Entity Framework Core
- Microsoft SQL Server
- AutoMapper
- HTML, CSS, Bootstrap
